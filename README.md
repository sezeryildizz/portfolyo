# Full-Stack Web Portfolyo Projesi

Bu proje, HTML5, CSS3, JavaScript, PHP ve MySQL teknolojileri kullanılarak geliştirilmiş dinamik bir kişisel web portfolyosudur.

## Projenin Amacı
Akademik ve profesyonel kariyerimde yeteneklerimi potansiyel işverenlere sergilemek amacıyla hazırlanmıştır. Sadece bir ödev değil, profesyonel bir dijital kimlik olarak tasarlanmıştır.

## Kullanılan Teknolojiler
*   **Ön Yüz (Frontend):** Semantik HTML5, Gelişmiş CSS (Flexbox/Grid), JavaScript (DOM Manipülasyonu, AJAX).
*   **Arka Yüz (Backend):** PHP, MySQL.
*   **Özellikler:** Duyarlı tasarım (Responsive), Karanlık Tema (Dark Mode), JS Form Doğrulama, Admin Paneli (Oturum Yönetimi - Session), Veritabanı destekli dinamik projeler ve iletişim mesajları.

## Kurulum
1. Veritabanı ayarları için `database.sql` dosyasını içe aktarın.
2. PHP yapılandırması için `includes/db_connect.php` (henüz oluşturulmadı) dosyasını kendi sunucunuza göre düzenleyin.
